# Markdown Viewer Enhanced

> 一款浏览器扩展，在浏览器中优雅地渲染 Markdown 文件，支持文件浏览器侧边栏、Mermaid 图表、代码高亮、KaTeX 数学公式、目录导航、多主题切换等功能。

---

## ✨ 功能亮点

### 🎨 多主题 & 代码高亮
- **三种页面主题** — 浅色 / 深色 / 跟随系统自动切换
- **15 种代码高亮主题**（含 auto）— Default Light/Dark Modern、GitHub、Monokai、VS 2015、Atom One Dark/Light、One Dark Pro、Dracula、Nord、Solarized Light/Dark、Tokyo Night，亮色暗色自由搭配
- 代码高亮主题可设置为"跟随页面主题"自动切换

### 📊 Mermaid 图表渲染
- 完美支持 Mermaid 语法，直接在浏览器中渲染流程图、时序图、甘特图、类图、状态图等
- 图表随页面主题自动切换亮色/暗色配色
- **点击图表弹窗放大预览**，支持鼠标滚轮缩放、拖拽平移、键盘快捷键（`+` / `-` / `0`）
- 适应窗口 / 重置缩放一键操作
- 支持复制 Mermaid 源码

### 💻 代码块增强
- 基于 highlight.js，支持 **180+** 种编程语言语法高亮
- 代码块顶部显示**语言标签**和**一键复制**按钮
- 可选**行号显示**，方便代码引用和阅读
- **diff 语法高亮**：自动识别 `+` / `-` 行，添加/删除行整行背景色
- 代码高亮主题独立于页面主题，亮色页面也能用暗色代码主题

### 🔢 KaTeX 数学公式
- 支持 LaTeX 语法的**行内公式**（`$...$`）和**块级公式**（`$$...$$`）
- 基于 KaTeX 高性能渲染，字体从 CDN 自动加载
- 智能区分数学公式与货币符号，避免误渲染

### 📑 目录导航
- 自动从文档标题提取生成**目录树**
- 支持**左侧 / 右侧**两种侧边栏布局
- 目录项支持**折叠/展开**子级（▸/▾ 按钮）
- 点击目录项**平滑滚动**跳转到对应章节
- 当前阅读位置**自动高亮**，滚动时实时追踪
- 支持 **URL hash 定位**，页面加载后自动跳转到锚点位置
- 可随时展开/折叠目录侧边栏
- 侧边栏支持**拖拽调整宽度**（180px - 600px）

### 📐 灵活的排版设置
- **字体大小**：12px - 24px 自由调节
- **行高**：1.2 - 2.0 宽松度可调
- **内容宽度**：600px - 1400px 自定义
- **正文字体**：系统默认 / 衬线体 / 等宽字体
- 排版变更实时预览

### 📁 文件浏览器侧边栏（仅 file:// 协议）
- 自动读取当前文件所在**目录的文件列表**
- **文件夹展开/折叠**：点击文件夹就地展开子目录内容，懒加载子目录数据
- **文件在新标签页打开**：点击文件自动在新标签页中打开
- **面包屑导航**：显示当前文件路径，点击上级目录可在新标签页打开
- 支持**排序方式**切换（按名称 / 大小 / 修改日期）以及升序 / 降序
- 支持**文件夹置顶**和**显示/隐藏隐藏文件**
- 自动过滤仅显示支持的 Markdown 文件格式
- 当前文件自动高亮标记
- 侧边栏**页签切换**：📁 文件浏览器 / ≡ 目录导航

### 🖼️ 图片增强
- 点击图片**弹窗放大预览**
- 图片支持**懒加载**，优化长文档性能
- 按 `Esc` 快速关闭预览

### 📝 Markdown 扩展语法
- ✅ **GitHub 风格告警块**：`> [!NOTE]` / `> [!TIP]` / `> [!IMPORTANT]` / `> [!WARNING]` / `> [!CAUTION]`，并额外支持 `> [!BLANK]`
- ✅ **任务列表**：`- [x]` / `- [ ]`，美观的复选框样式
- ✅ **脚注**：基于 marked-footnote 扩展
- ✅ **定义列表**：`Term` + `: Definition` 格式（PHP Markdown Extra 风格）
- ✅ **表格**：增强表头渐变样式、行悬浮高亮、sticky 表头、圆角外框
- ✅ **外部链接**自动在新标签页打开

### 🔧 工具栏 & 快捷操作
| 操作 | 说明 |
|------|------|
| 📑 目录 | 切换目录侧边栏显示/隐藏 |
| 🌓 主题 | 快速在浅色/深色主题间切换 |
| 📝 源码 | 查看 / 返回 Markdown 原始源码 |
| ⚙️ 设置 | 打开扩展高级设置页面 |
| 🔃 刷新 | 重新加载当前页面 |

- 滚动超过 300px 自动显示**浮动回到顶部**按钮
- `Esc` 键关闭图片预览和 Mermaid 图表预览

### ⚙️ 设置系统
- **Popup 快捷面板**：点击扩展图标即可快速调整主题、字体、宽度、目录、Mermaid/KaTeX 开关
- **Options 高级设置页**：完整的排版配置、代码主题选择（含实时预览）、行号、行高等高级选项
- 设置通过 `storage.sync` 同步，跨设备保持一致
- 设置变更实时推送到所有已打开的 Markdown 标签页
- 一键恢复默认设置

## 📋 支持的文件格式

| 扩展名 | 说明 |
|--------|------|
| `.md` | Markdown 文件 |
| `.mdc` | Markdown 组件文件 |
| `.markdown` | Markdown 文件 |
| `.mkd` | Markdown 文件 |
| `.mdown` | Markdown 文件 |
| `.mdtxt` | Markdown 文本文件 |
| `.mdtext` | Markdown 文本文件 |

本地 `file://` 支持以上全部扩展名；远程 `http://` / `https://` 当前支持 `.md`、`.mdc`、`.markdown`。

## 🚀 快速开始

1. **安装插件**：从插件市场安装本插件
2. **打开 Markdown 文件**：在浏览器中直接打开 `.md` 文件
3. **自动渲染**：插件自动检测并渲染 Markdown 内容
4. **自定义设置**：点击扩展图标，调整主题、字体等偏好

### 访问本地文件

如果需要渲染本地 Markdown 文件，请确保已启用插件的文件访问权限：
1. 进入浏览器的扩展管理页面
2. 找到 **Markdown Viewer Enhanced**
3. 点击「详情」
4. 开启「允许访问文件网址」

## 🛠️ 技术栈

- **Markdown 解析**：[Marked.js](https://github.com/markedjs/marked)
- **图表渲染**：[Mermaid](https://github.com/mermaid-js/mermaid)
- **代码高亮**：[highlight.js](https://github.com/highlightjs/highlight.js)
- **数学公式**：[KaTeX](https://github.com/KaTeX/KaTeX)
- **脚注扩展**：[marked-footnote](https://github.com/bent10/marked-extensions)
- **安全过滤**：[DOMPurify](https://github.com/cure53/DOMPurify)

---

## 🔒 隐私声明

- 本插件 **不会** 收集任何用户数据
- 本插件 **不会** 向任何服务器发送信息（KaTeX 字体 CDN 加载除外）
- 所有设置数据仅存储在浏览器本地（`storage.sync`）
- 仅需要 `activeTab`、`storage`、`tabs`、`scripting` 权限（后两项用于本地文件浏览器功能）
